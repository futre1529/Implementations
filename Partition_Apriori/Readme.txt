AIM _ To find the frequent itemsets from the given dataset

Algorithm - Partition the dataset into smaller parts and apply apriori on each 
partition and then scan the local frequent itemsets with respect to the complete
dataset.

Sample Input - "partition.txt"

Sample Output - 

Frequent itemsets - {1},{2},{3},{4},{5},{1,2},{1,3}{1,5},{2,3},{2,4},{2,5},{1,2,3},{1,2,5}
